Cheatsheet mec-2

If you want to know more about *mec2* i recommend you to read more about the documentation https://www.gislain1993.github.io/mec2-Dokumentation/

Properties Comment Required properties Optional properties
Nodes:[] Create a node by giving the required properties x-cordinate, y-cordinate, id base, idloc
Constraints:[] defines the relationship between two nodes. i.e. in order to define a constraint, two nodes must first of all be defined. id, P1, P2 drive, reference, sequence
Loads:[] we have two main types of loads: force and springs P, Wo, id, mode, type -
Views:[] can be used to make different properties of the model or single elements of the model visible. show,of,as,Dt vector, trace, info, chart
Shapes:[] to stylize elements with geometric forms or images type, p beam, bar, image, slider,wheel
